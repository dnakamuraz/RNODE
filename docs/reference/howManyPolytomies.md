# How many Polytomies in a tree

This function will a single value for the number of polytomies in a tree

## Usage

``` r
howManyPolytomies(phy)
```

## Arguments

- phy:

  A phylogenetic tree in the class "phylo"

## Value

Returns single value for the number of polytomies in a tree

## Author

Barb Banbury

## Examples

``` r
tree <- rtree(10)
howManyPolytomies(tree)
#> Error in howManyPolytomies(tree): could not find function "howManyPolytomies"
```
