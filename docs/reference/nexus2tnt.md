# nexus2tnt

Converts a NEXUS matrix into TNT format while preserving header data
type and ordered character structures.

## Usage

``` r
nexus2tnt(input_file, output_file)
```

## Arguments

- input_file:

  Path to the input NEXUS file.

- output_file:

  Path to the output TNT file.
