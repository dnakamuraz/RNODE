# tnt2nexus

Converts a TNT matrix into NEXUS format while preserving header data
type and ordered character structures.

## Usage

``` r
tnt2nexus(input_file, output_file)
```

## Arguments

- input_file:

  Path to the input TNT file.

- output_file:

  Path to the output NEXUS file.
